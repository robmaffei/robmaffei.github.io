Excel

Automated data loading and transformation with Power Query, interactive dashboards with Pivot Tables and Pivot Charts.

Here we have an Excel dashboard summarizing weekly air quality data (PM10) in Slovakia (data from the Slovak Hydrometeorological Institute).

Since the worksheet is embedded in the webpage and hosted on OneDrive it has some limitations. For instance, automated updates are currently not supported. To update the worksheet you need to open it in the desktop app.

Nonetheless this example showcases the following features:

  • Automated data acquisition and cleaning via Power Query;
  • Interactive dashboard with Pivot Tables and Charts.

Feel free to try out the dashboard.

This table has been designed to provide a quick overview of an indicator: In the middle we have daily and weekly averages, while on the right are highlighted (thanks to conditional formatting) some readings that might require our attention.

An in-depth report requires a different structure and more data (e.g. historic readings, additional indicators), however, this work could represent a starting point or a dedicated page for a more extensive analysis.
